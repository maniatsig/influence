What is this?

This is a web analytics tool which combines Google Analytics data of a real website with population data, in order to help an analyst understand the influence his website has per greek city or region for a specific period of time. By default the tool shows Google Merchandise Store's influence in Greece for year 2017.

How it works?

This tool divides the number of visitors for each city or region with population and gives a percentage of website's awareness.
Notice: This tool works only if a website has greek audience.

I monitor my website with Google Analytics. How can I use it?

In order to use this tool with your own GA data you need to:
  1. Load to your Google Analytics the Custom Report Users per city
  2. Set the time period you want to check (a whole year is recommended).
  3. Add as asecondary attribute thw City Identifier (under Users).
  4. Export the report to Google Spreadsheet.
  5. Share the spreadsheet and copy the shared link.
  6. Select the "Load Data" menu of the tool and paste the link.
